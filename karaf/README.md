# Karaf module for Puppet #

This module installs Karaf.

## Description

This module can install Karaf in a simple way.

Currently supports:

* Ubuntu 14.04.

## Usage

General install with:

     class { 'karaf':
     	   user => 'huluvu424242',
	   karafVersion => '3.0.1',
	   tmpDir => '/tmp',
     }
